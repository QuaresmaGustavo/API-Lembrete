## Sobre

    # Este projeto é uma lista de tarefas (to-do list) que permite inserir e excluir lembretes. Os lembretes inseridos serão apresentados em ordem cronológica. 

##Tecnologias usadas

    # FrontEnd
        - React + vite
        - Styled components
        - Google Fonts
        
    # Backend
        - EntityFrameWorkCore
        - EntityFrameWorkCore Tool
        - EntityFrameWorkCore Design
        - Pomelo MySQL

    # Banco de Dados
        - MySQL
